# Codex Guidelines

## Ziel

Diese Datei beschreibt, wie Codex beim Entwickeln von Humanbase eingesetzt werden soll.

Codex soll helfen, schneller zu entwickeln, aber nicht unkontrolliert große Änderungen machen.

## Grundregel

Codex ist ein Coding-Assistent, nicht der alleinige Architekt.

Der Mensch entscheidet:

- Produktlogik
- Prioritäten
- Architektur
- Review
- Merge
- Veröffentlichungen

Codex hilft bei:

- Code schreiben
- Dateien erstellen
- Bugs finden
- Refactoring
- Tests
- Dokumentation
- Erklärungen
